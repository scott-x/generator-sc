# sc-react-cli

## introduction
It's a cli for react.

## how to use?

### 1. installation
```
npm i react-generator-sc -g

```
### 2. init project
```
# default myApp
sc i [yourProjectName]
```
## command
```
sc g c yourComponentName
sc g p yourPageName
```
## Test
```
npm run start
```
